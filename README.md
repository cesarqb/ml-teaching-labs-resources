# ML Teaching Labs Resources
Este repositorio contiene materiales estructurados de enseñanza y laboratorios prácticos utilizados en cursos introductorios de Ciencia de Datos y Machine Learning.
El objetivo es proporcionar ejemplos prácticos y progresivos que permitan a estudiantes y profesionales comprender los fundamentos técnicos de manera aplicada.

---

## Público Objetivo
- Estudiantes universitarios en carreras STEM  
- Profesionales que inician en Ciencia de Datos  
- Personas que buscan reforzar fundamentos en Python, R y Machine Learning  

---

## Contenidos Cubiertos
- Introducción a Python para análisis de datos  
- Introducción a R para análisis estadístico  
- Análisis Exploratorio de Datos (EDA)  
- Fundamentos de Machine Learning  
- Ejercicios prácticos con datasets reales  
- Notebooks guiados paso a paso
